Question-1: What are the different ways to select an element in the DOM?

The different ways to select an element in DOM is: 1. getElementById("id") 2. getElementsByClassName("className") 3. getElementsByTagName("tagName") 4. querySelector("selector") 5. querySelectorAll("selector").

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Retrieves the HTML content inside an element
innerText: Retrievesthe visible text inside an element
textContent: Retrievesthe full text content inside an element, including hidden elements

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a parent element listens for events on its child elements instead of adding event listeners to each child.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process in the DOM where an event starts from the target element and bubbles up to its parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element document.createElement(tagName) is used.
To add an element as last child of parent document.body.appendChild(newDiv) is used.
to add multiple elements parent.append(newDiv, "Additional text") is used.
To remove an element newDiv.remove() is used.
To remove a child element parent.removeChild(child) is used.
